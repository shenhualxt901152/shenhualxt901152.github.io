PasteKeyboardX

PasteKeyboardX is a custom keyboard for saving reusable text clips and inserting them from the keyboard.

Features

  • Save short clips, links, replies, and temporary text.
  • Insert saved clips from the custom keyboard.
  • Import clipboard text only after tapping the keyboard clipboard button.
  • Optionally sync clips through the user's iCloud Private Database.
